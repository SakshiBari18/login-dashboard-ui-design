# 🔐 Login Page UI/UX Prototype

## 📌 Overview

This project is an interactive UI/UX prototype designed in Figma as part of my UI/UX learning journey. The prototype consists of multiple connected screens with navigation, focusing on creating a clean, modern, and user-friendly authentication experience.

The goal of this project was to practice interface design, user flow creation, visual hierarchy, and Figma prototyping while building a realistic login experience.

---

## 🎯 Project Objectives

* Design a modern and intuitive login interface
* Create seamless navigation between screens
* Improve visual hierarchy and layout organization
* Practice UI/UX design fundamentals
* Gain hands-on experience with Figma prototyping

---

## ✨ Features

* Modern and clean user interface
* Multi-screen user flow
* Interactive navigation between pages
* Consistent typography and spacing
* User-friendly layout design
